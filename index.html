使用說明  ·  User Guide
第一步：安裝 NDI Tools
Windows 和 Mac 都需要安裝。
1. 前往 ndi.video/tools 下載 NDI Tools
2. 安裝時勾選所有元件（特別是 Adobe Creative Cloud Plugin）
3. 安裝完成後重新啟動電腦
第二步：在 After Effects 開啟 NDI 輸出
Windows：
1. 開啟 After Effects 專案
2. 上方選單 → Edit → Preferences → Video Preview
3. 勾選 ☑ Enable Mercury Transmit（啟用 Mercury 傳輸）
4. 在下方列表中勾選 ☑ NDI output
5. 按 OK 確認

Mac：
1. 開啟 After Effects 專案
2. 上方選單 → After Effects → Settings → Video Preview
3. 勾選 ☑ Enable Mercury Transmit
4. 在下方列表中勾選 ☑ NDI output
5. 按 OK 確認
第三步：開始串流
1. 在 AE 中按下播放 ▶（或拖動時間軸）
2. AE 會自動透過網路發送 NDI 訊號
3. 打開舞臺預覽工具，點擊「🔴 連接 NDI → 所有面板」
4. 在彈出的列表中選擇你的電腦名稱（例如 ULTRA9-285K (Adobe After Effects)）
5. 面板會自動顯示 AE 畫面，自動對應每個 LED 區域
常見問題
Q：找不到 NDI 來源？
→ 確認 AE 的 Video Preview 設定中 Mercury Transmit 和 NDI output 都有勾選
→ 確認 AE 有在播放狀態（靜止時不會持續送訊號）
→ 確認兩台電腦在同一個網路（同一個 Wi-Fi 或有線網路）

Q：切換到其他視窗後畫面就斷了？
→ AE 失去焦點時會暫停渲染。建議把 AE 和預覽工具放在不同螢幕上
→ 或到 AE 設定：Edit → Preferences → Display 確認背景渲染選項

Q：畫面解析度很低？
→ 檢查 AE 預覽視窗右下角的 Resolution 設定
→ Full = 最高畫質，Half = 1/2，Third = 1/3
→ 本工具會自動縮放對應，但越高解析度越吃效能

Q：支援哪些畫布尺寸？
→ 10912 × 2024：含樂手 LED（轉台背面小面板）
→ 8800 × 2024：純主牆（SKY + LED-1~4）
→ 工具會根據收到的畫面比例自動偵測並切換佈局
